## 🚀 Installation & Développement

### Prérequis
- Node.js 18+ 
- npm ou yarn
- Git

### Installation

```bash
# Cloner le repository
git clone https://github.com/FlotchNotch/Portfolio-V1.git
cd Portfolio-V1

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

### Scripts Disponibles

```bash
npm run dev          # Démarrer en mode développement
npm run build        # Créer le build de production
npm run start        # Démarrer le serveur de production
npm run lint         # Lancer ESLint
npm run type-check   # Vérifier les types TypeScript
```

## 🔧 Personnalisation

Pour adapter ce portfolio à vos besoins :

1. **Contenu** : Modifiez les textes dans les composants
2. **Couleurs** : Ajustez la palette dans `tailwind.config.js`
3. **Animations** : Personnalisez les animations Framer Motion
4. **Projets** : Ajoutez vos projets dans le composant Projects
